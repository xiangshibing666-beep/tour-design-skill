# 旅游安全性分析指南（Safety Analysis Guide）

> 本指南定义「旅游安全性分析」文档的调查方法、数据来源、风险矩阵与产出结构。
> 分析必须结合「既往气候 / 灾害报告」+「未来逐日预报」+「实时预警」+「风俗习惯」四个维度，给出可执行的应对预案，而非空泛结论。

## 一、数据来源与时效说明

| 类别 | 来源 | 可靠性 |
|------|------|--------|
| 气候统计 | 省市气候中心《XX 年 X 月气候评价》、中国气象局年鉴 | 高 |
| 历史灾害 | 央视 / 人民网 / 百度百科（滑坡、洪水、闭园案例） | 高 |
| 趋势预测 | 省市气象局《盛夏 / 月度气候趋势预测》 | 中（趋势级） |
| 逐日预报 | 中央气象台·中国天气网 15 天、IP.cn、tianqi.com | 中（8–15 天波动大） |
| 实时预警 | 中央气象台暴雨 / 地灾预警、省水利厅山洪预警、自然资源部地灾预警 | 高（实时） |
| 景点脆弱性 | 景区官方公告、抖音 / 新闻闭园 / 索道停运报道 | 中高 |
| 风俗习惯 | 官方文旅、地方志、权威游记 | 中高 |

> ⚠️ **时效警告**：中国天气网明确标注 8–15 天预报为「集合多家数值模式客观产品，未经预报员订正，反映趋势、不确定性较大」。行程后段数据仅供参考，必须提示用户出发前 3 天复查。

## 二、调查清单（WebSearch 主题）

1. **历史气候**：目的地省市「X 月气候评价」「主汛期」「暴雨频率」「历史灾害」（滑坡 / 洪水 / 闭园）
2. **趋势预测**：当地气象局「盛夏 / 月度预测」——注意与当前实况是否矛盾（预测偏旱但实况暴雨，须以实况为准）
3. **实时预警**：「中央气象台 暴雨预警」「XX 省 地质灾害预警」「XX 山洪预警」——抓最新发布日期与等级
4. **台风**：编号台风路径与影响时段，判断是否覆盖行程日期（台风多在华南 / 华东，内陆影响递减）
5. **逐日预报**：`weather.com.cn/weather15dn/【区号】.shtml` 抓天气状况；用 IP.cn / tianqi.com 补全最高 / 最低温（中国天气网 WebFetch 常缺温度）
6. **景点脆弱性**：各景点「暴雨 关闭」「索道停运」「闭园」历史案例（抖音 / 新闻 / 官方公告）——证明风险真实存在
7. **风俗习惯**：宗教禁忌、饮食禁忌、节庆、礼仪（官方文旅 / 地方志 / 游记）

## 三、各景点风险矩阵模板

| 景点 | 暴雨脆弱点 | 风险等级 | 说明 |
|------|-----------|----------|------|
| 【山区/山顶】 | 雷击、大风、雷暴闭园、索道停运、地质灾害 | 🔴 高 | 有真实闭园/滞留先例 |
| 【涉水景区】 | 山洪致水位暴涨、游船停运、步道封闭 | 🟠 中高 | 涨水时限流/封闭 |
| 【山地村寨】 | 边坡、观景台、进寨道路滑坡/落石 | 🟠 中高 | 连日雨后不稳定 |
| 【傍水古镇】 | 河岸/低洼/桥面，洪水期水位上涨 | 🟡 中 | 河边活动需谨慎 |
| 【瀑布】 | 上游来水过大临时闭园；水帘洞遇大雨关闭 | 🟡 中 | 丰水期壮观但极端限流 |

风险等级：🔴 高 / 🟠 中高 / 🟡 中

## 四、分析产出结构

1. **数据来源与时效说明**
2. **既往报告**：气候统计 + 历史灾害案例 + 趋势预测 vs 当前实况矛盾
3. **未来预报**：逐城市逐日温度 / 降雨表 + 台风影响评估
4. **各景点风险矩阵**
5. **综合安全结论**：是否整天暴雨（多为午后雷阵雨）、今年汛期强弱、行程是否天然避开最坏天气
6. **应对预案**：分景点 Plan B + 装备清单 + 保险 + 赶路提示
7. **一句话总结**

## 五、风俗习惯要点（通用，按目的地增删）

- **宗教场所**（寺庙 / 清真寺 / 教堂）：着装得体、禁喧哗、拍照前确认、绕行方向
- **少数民族地区**（苗 / 侗 / 藏 / 维吾尔等）：敬酒礼仪、禁忌话题、服饰与头像尊重、长桌宴规矩
- **饮食**：辣度 / 生食 / 常见过敏原、肠胃适应建议、自来水是否可饮
- **节庆**：是否撞上大型活动（人流暴涨 / 住宿涨价 / 交通管制）
- **方言 / 礼仪**：还价习惯、小费文化、拍照征询

## 六、铁律

- 不编撰天气 / 灾害数据；预报标注时效；实时预警优先于历史统计
- 结论给「可控但需主动避险」的 actionable 建议，既不吓人也不轻描淡写
- 每个风险点必须配套 Plan B（替代景点 / 改期 / 保险）
